# 🎲 Formiga Lúdica MVP

MVP de uma plataforma web para locadora de jogos, com catálogo online, cadastro de jogos e recomendações inteligentes com IA.

---

## Sobre o projeto

O Formiga Lúdica MVP é um sistema web desenvolvido para auxiliar na gestão de uma locadora de jogos de tabuleiro.

O objetivo é automatizar processos atualmente realizados manualmente, oferecendo uma melhor experiência aos clientes e facilitando o gerenciamento do negócio.

---

# Funcionalidades previstas 

- [x] Cadastro manual de jogos
- [x] Catálogo online
- [ ] Recomendação inteligente de jogos com IA
- [ ] Importação de jogos via API da Ludopedia
- [ ] Carrinho de locação
- [ ] Controle de disponibilidade por período
- [ ] Sistema de pedidos
- [ ] Avaliação de jogos
- [ ] Programa de fidelidade

---

# Tecnologias utilizadas 

- PHP
- MySQL
- JavaScript
- HTML5
- CSS3

---

# Inteligência Artificial 

O projeto contará com um agente de IA responsável por recomendar jogos aos clientes com base em informações como:

- Quantidade de jogadores;
- Faixa etária;
- Ocasião;
- Nível de experiência;
- Tempo disponível para jogar.
